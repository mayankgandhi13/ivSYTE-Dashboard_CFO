# ivSYTE Diagnostics — CFO Investor Dashboard

> CFO investor dashboard for **ivSYTE Diagnostics**, a hypothetical multi-cancer early detection company using blood-based biomarker imaging AI. Built as part of the BIOT 5219 Team Business Pitch Project at Northeastern University.
🔗 **Landing Page:** [mayankgandhi13.github.io/ivSYTE](https://mayankgandhi13.github.io/ivSYTE/landing.html)
🔗 **Live Dashboard:** [mayankgandhi13.github.io/ivSYTE-Dashboard_CFO](https://mayankgandhi13.github.io/ivSYTE)

---

## About

ivSYTE Diagnostics is a hypothetical biotech company developing a blood-based multi-cancer early detection diagnostic using VitaSpectra AI — a proprietary biomarker imaging engine covering 200+ cancer type signatures.

| Metric | Value |
|---|---|
| Total Funding Ask | $120M (Series A) |
| Anchor Investor Ask | $40M |
| Angel Seed Raised | $75M |
| VC Loan Secured | $100M |
| Sensitivity | 91% (VitaSpectra AI) |
| Specificity | 97% |
| FDA PMA Target | 2025 Q4 |
| Break-Even | 2029 |
| Projected IRR | ~34% |

---

## Dashboard Sections

- **KPI Strip** — Key financial and clinical metrics
- **Business Timeline** — Funding milestones from founding (2018) to commercial launch (2028)
- **Market Opportunity** — TAM / SAM / SOM ($B), 2023–2033
- **Revenue Forecast vs. Burn** — Projected ramp with break-even at 2029
- **Competitive Landscape** — Sensitivity vs. price bubble chart (vs. Grail Galleri, NGS Liquid Biopsy, Imaging CT/MRI, Standard Panel)
- **Product Positioning Radar** — 6-dimension comparison vs. market average
- **Use of Proceeds** — $120M Series A donut chart breakdown
- **Milestone Tracker** — Path to value with completion status
